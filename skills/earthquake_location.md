# Earthquake Location

Use this skill when you want to estimate an event origin from picked phases, station metadata, and travel-time information.

## Typical workflow

1. Confirm which stations and phases are available.
2. Inspect the quality of the P and S picks.
3. Run the location workflow with a consistent coordinate reference.
4. Review residuals and map output for outliers.

## Useful inputs

- Station coordinates
- Picked arrival times
- Phase labels
- Optional event depth bounds

## Output to inspect

- Origin time
- Latitude and longitude
- Depth
- Residual statistics
- Location map or summary table
