# Continuous Monitoring

Use this skill when you need to inspect live or recent waveform data over a time window.

## Typical workflow

1. Choose the monitoring region or station set.
2. Select the time window, such as the last few hours.
3. Review the incoming traces for new events or anomalies.
4. Export plots or summaries for follow-up analysis.

## Useful checks

- Recent amplitude spikes
- Missing channels
- Time coverage gaps
- Repeating event patterns
- Whether the station set is stable

## Expected output

- Monitoring summary
- Waveform preview
- Detected candidate events
- Notes about data quality
