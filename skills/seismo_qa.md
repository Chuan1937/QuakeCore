# Seismo QA

Use this skill when the user asks a conceptual earthquake or seismology question.

## Typical scope

- Basic earthquake terminology
- Wave types and phase names
- Picking and location concepts
- Data format questions
- Workflow explanations

## Best practice

- Answer directly and keep the explanation tied to the user question.
- If the question depends on the current dataset, ask for the file or show the relevant context first.

## Expected output

- Concise explanation
- Practical next step if the user is working with data
