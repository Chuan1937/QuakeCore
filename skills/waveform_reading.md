# Waveform Reading

Use this skill when you need to inspect one trace or a small group of traces directly.

## Typical workflow

1. Load the current waveform or selected channel.
2. Check the start time, sampling rate, and trace length.
3. Read the amplitude pattern before and after the event.
4. Compare neighboring channels if the signal looks unusual.

## Useful checks

- Baseline drift
- Missing samples
- Clipping or spikes
- Event onset timing
- Station or channel differences

## Expected output

- Waveform summary
- Trace metadata
- Visual plot or excerpt
- Notes about anomalies
