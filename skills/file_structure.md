# File Structure

Use this skill when you need a quick overview of a seismic file before deeper analysis.

## What to check

- File type and format
- Number of traces or records
- Sampling interval and sample count
- Channel or station identifiers
- Available headers or metadata fields

## Common questions

- Does the file contain one station or many?
- Are the traces aligned in time?
- Is the metadata sufficient for phase picking or location?
- Are there obvious gaps, clipping, or malformed headers?

## Practical output

- File summary
- Trace list or channel map
- Header field inspection
- Warnings about missing metadata
