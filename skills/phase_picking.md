# Phase Picking

Use this skill when you want to identify P and S arrivals from waveform data.

## Typical workflow

1. Inspect the waveform window and choose an appropriate time range.
2. Confirm the station, channel, and sampling rate.
3. Run automatic phase picking if available.
4. Review the picks visually and adjust obvious mistakes.

## What matters

- Signal-to-noise ratio
- Trace alignment and gaps
- Saturation or clipping
- Whether the waveform contains one event or multiple arrivals

## Output to inspect

- Picked P and S times
- Confidence or quality scores
- Trace preview with markers
- Notes about ambiguous or low-quality picks
