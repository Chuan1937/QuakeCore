# Map Plotting

Use this skill when you want to visualize stations, events, or locations on a map.

## Typical workflow

1. Gather coordinates for stations and events.
2. Choose a plot extent that fits the region.
3. Add labels, legends, and a scale reference if needed.
4. Export the map for reporting or debugging.

## Useful inputs

- Station coordinates
- Event epicenter or hypocenter
- Region bounds
- Optional tectonic or geographic context

## Expected output

- Location map
- Station and event markers
- Legends or annotations
- A file path to the generated figure
