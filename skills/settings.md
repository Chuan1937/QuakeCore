# Settings

Use this skill when the user asks about configuration, environment settings, or LLM setup.

## Typical scope

- Provider selection
- Model name configuration
- API keys and base URLs
- Local config file behavior

## Best practice

- Prefer the simplest configuration that works for the current environment.
- Do not expose secrets in logs or examples.

## Expected output

- Current configuration summary
- Save/load guidance
- Notes about defaults and overrides
